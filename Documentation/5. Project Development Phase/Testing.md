# Testing

Testing was conducted to ensure that the House Hunt application works
correctly, securely, and efficiently according to the defined
requirements.

## Functional Testing

Functional testing was performed to verify that all system features
operate as expected.

The following functionalities were tested:
- User registration and login
- Authentication and authorization
- Protected route access
- Display of apartment listings
- Property search and filtering
- Viewing apartment details
- Dashboard access and logout

All forms were tested for validation, including incorrect inputs and
empty fields.

## Performance Testing

Performance testing evaluated the responsiveness and efficiency of the
application.

The following aspects were tested:
- Page load time
- Rendering speed of apartment listings
- Authentication response time
- Navigation between pages

The application showed fast loading and smooth navigation due to React’s
optimized rendering.

## Security Testing

Security testing ensured protection of user data and restricted access.

The following checks were performed:
- Secure login and signup using JWT
- Prevention of unauthorized dashboard access
- Protected routes implementation
- Proper logout and session handling

The system successfully passed all security checks, ensuring reliable
user data protection.
