# Customer Journey Map

The customer journey map represents the complete flow of a user interacting with
the House Hunt platform, from initial access to contacting a property owner.

## User Journey Stages

### 1. Awareness
The user becomes aware of the House Hunt platform through online sources or
recommendations and visits the website.

### 2. Registration & Login
- New users create an account using signup
- Existing users log in using secure authentication

### 3. Property Browsing
- User views available rental listings
- Properties are displayed with price, location, and basic details

### 4. Search & Filtering
- User applies filters such as location, budget, and property type
- The system dynamically updates results

### 5. Property Details View
- User selects a property
- Views detailed description, amenities, images, and rent details

### 6. Contact Owner
- User contacts the property owner directly
- Clarifies doubts or schedules a visit

### 7. Decision Making
- User compares shortlisted properties
- Makes a final rental decision

This journey ensures a smooth, transparent, and efficient rental search
experience for users.
