# Empathy Map Canvas

The empathy map helps understand the user’s needs, emotions, and pain points
while using traditional ordering systems.

## Thinks
- Wants fast and easy ordering
- Expects accurate order details
- Prefers digital solutions

## Feels
- Frustrated with long waiting times
- Confused by manual processes
- Worried about order errors

## Says
- “Ordering should be quick”
- “I want real-time updates”
- “Manual systems waste time”

## Does
- Avoids crowded places
- Searches for online alternatives
- Uses mobile apps for convenience

## Pain Points
- Delays in order processing
- No tracking system
- Manual errors

## Gains
- Quick ordering
- Real-time notifications
- Improved user experience
